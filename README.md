An application for connecting freelancers and clients.

##  How to Run
1. Navigate to the `/mvp` folder.
2. Run `npm install` in both `client` and `server` folders.
3. **To start the development environment:**
   * In the `/client` folder, run: `npm run dev`
   * In the `/server` folder, run: `npm start` (or `node index.js`)
